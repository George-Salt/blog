HTML - это язык разметки, который используется для формирования содержимого веб-страницы в браузере.

HTML-документ это файл с разметкой страницы, имеет расширение .html, напоминает текстовый файл - в нем много строчек с буквами и символами (в основном < > ! /). HTML-документ состоит из множества элементов. Каждый элемент имеет свой тег (идентификатор). По тегу можно отличить текст от картинки или блочного контейнера.

Структура HTML-документа

Как правило, HTML-документ, написанный на пятой версии языка начинается строчкой:


        <!
        DOCTYPE
         html
        >

Он определяет тип документа и необходим для того, чтобы браузер понимал, как следует интерпретировать текущую веб-страницу, так как разметка может быть написана на разных версиях HTML или на XHTML.
